# 🔐 Security Policy

## Supported Versions

We actively support the latest version of momAid backend.

| Version | Supported |
|--------|-----------|
| latest | ✅ |

---

## Reporting a Vulnerability

If you discover a security vulnerability:

- Do NOT open a public issue
- Email: belammuia0@gmail.com
- We will respond within 48–72 hours

---

## Scope

This project includes:
- Authentication system (JWT, Google OAuth)
- Video upload system
- Comment system
- Background tasks (Celery)

---

## Out of Scope

- Third-party services (Google, Cloudinary, Redis)
